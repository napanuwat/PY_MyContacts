💡 UX Tips ที่ควรทำใน Excel:
• Row 1 + Column B = Freeze Pane → เลื่อน scroll แล้วยังเห็น Header และชื่อเสมอ
• Conditional Formatting: แต่ละ Row สีอ่อนตามทีม (Core Banking = Indigo, PMO = Orange ฯลฯ)
• Data Validation (Dropdown) ที่ Column ทีม, โปรเจค, Role → ป้องกัน typo และ inconsistency
• Column A (ContactID) และ M-N = Lock (Protected) ไม่ให้แก้ไขโดยตรง
• ใส่ VBA Macro ปุ่ม "🔄 Sync Now" ใน Ribbon และ Sheet Sync

Section 04
VBA Macro — ตัวอย่างโค้ด Sync (Option B)
ถ้าเลือก Option B ต้องเขียน VBA Macro ในปุ่ม "Sync Now" — นี่คือ structure ของโค้ดที่ต้องใช้
' VBA Module: MyContacts_Sync.bas
' ================================================================
' ปุ่ม "Sync Now" → เรียกใช้ Macro นี้
' ================================================================

Sub SyncContacts()
Dim http As Object
Dim jsonBody As String
Dim ws As Worksheet

Set ws = ThisWorkbook.Sheets("Contacts")
Set http = CreateObject("MSXML2.XMLHTTP")

' 1. สร้าง JSON จาก Excel rows
jsonBody = BuildJSON(ws)

' 2. POST ไปที่ Python API
http.Open "POST", "http://localhost:5000/api/sync", False
http.setRequestHeader "Content-Type", "application/json"
http.send jsonBody

' 3. รับ Response และอัพเดต Sync Status Column
If http.Status = 200 Then
Call UpdateSyncStatus(ws, "✅ Synced")
MsgBox "Sync สำเร็จ! " & Now(), vbInformation
Else
Call UpdateSyncStatus(ws, "❌ Error")
MsgBox "Sync ไม่สำเร็จ: Status " & http.Status, vbCritical
End If
End Sub

' ============================================================
' Python API endpoint ที่ต้องสร้าง (Flask)
' POST /api/sync → รับ JSON → save to SQLite → return updated data
' GET /api/contacts → Web ใช้ดึงข้อมูล
' ============================================================
